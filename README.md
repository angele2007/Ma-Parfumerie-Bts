# Ma Parfumerie

Application de gestion de stock pour une parfumerie.

## 📋 Contexte

Projet réalisé dans le cadre du BTS Développement d'Applications Informatiques (DAI) — Togo.

## 🛠️ Technologies

- **Langage** : Java
- **Interface** : Swing
- **Base de données** : MySQL
- **IDE** : NetBeans

## ✨ Fonctionnalités

- Gestion des produits
- Gestion des fournisseurs
- Entrées et sorties de stock
- Facturation
- Impression des factures
- Recherche de produits

## 🚀 Installation

Ouvrir le projet dans NetBeans, configurer la connexion MySQL, puis lancer l'exécution.
